# 快速入门指南

本指南将帮助你在 10 分钟内完成 WizNote 到 Obsidian 的迁移。

## 前置准备

### 1. 检查系统环境

```bash
# 检查 Python 版本（需要 3.6+）
python3 --version

# 如果没有安装 Python，请先安装：
# macOS: brew install python3
# Ubuntu: sudo apt-get install python3
```

### 2. 安装依赖

```bash
# 克隆项目
git clone https://github.com/WardLu/wiznote-to-obsidian.git
cd wiznote-to-obsidian

# 安装 Python 依赖（仅在线下载需要）
pip3 install requests markdownify
```

## 快速迁移（3 步完成）

### 步骤 1: 从 WizNote 下载笔记

```bash
cd WizNote_Migration
python3 wiz_to_obsidian.py
```

**操作：**
1. 输入 WizNote 邮箱/账号
2. 输入密码（隐藏输入）
3. 等待下载完成

**输出：** `obsidian_export/` 目录

### 步骤 2: 优化 Markdown 格式

```bash
cd ../tools
export WIZNOTE_VAULT_DIR=~/ObsidianVault  # 设置你的 Vault 目录
python3 wiznote_to_obsidian.py --all
```

**功能：**
- 检查语法问题
- 修复格式错误
- 转换为 WikiLinks
- 修复图片路径

### 步骤 3: 迁移附件（可选）

```bash
# 迁移附件文件
python3 wiznote_to_obsidian.py --migrate-attachments

# 为笔记添加附件链接
python3 wiznote_to_obsidian.py --link-attachments
```

## 验证结果

### 1. 在 Obsidian 中打开 Vault

```bash
# 打开 Obsidian
# 选择 "Open folder as vault"
# 选择你的 Vault 目录（~/ObsidianVault）
```

### 2. 检查迁移结果

- [ ] 笔记文件是否正确显示
- [ ] 图片是否能正常显示
- [ ] 链接是否能正常跳转
- [ ] 附件是否能正常打开

### 3. 查看统计报告

```bash
python3 wiznote_to_obsidian.py --report
```

## 常见问题快速解决

### Q1: 登录失败？

**解决方案：**
1. 确认账号密码正确
2. 检查网络连接
3. 尝试在 WizNote 客户端中登录

### Q2: 图片显示不出来？

**解决方案：**
1. 检查图片目录是否存在
2. 运行 `--images` 修复图片路径
3. 确认图片文件没有被删除

### Q3: 找不到某个笔记？

**解决方案：**
1. 检查输出目录：`obsidian_export/`
2. 查看下载日志
3. 确认笔记在 WizNote 中存在
